⛧ Configuração Supabase ⛧
Para jogar online entre máquinas diferentes, você precisa de um projeto Supabase gratuito.
COMO CRIAR SEU PROJETO (grátis):
1. Acesse supabase.com e crie uma conta
2. Clique em "New Project" → dê um nome (ex: op-mesa)
3. Vá em SQL Editor e execute:
CREATE TABLE mesa_state (path text PRIMARY KEY, data jsonb, updated_at timestamptz DEFAULT now()); 4. Em Table Editor → mesa_state → Authentication, desative o RLS
5. Vá em Project Settings → API e copie a URL e a anon key
