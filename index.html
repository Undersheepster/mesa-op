⛧ Configuração Firebase ⛧
Para jogar online entre máquinas diferentes, você precisa de um projeto Firebase gratuito.
COMO CRIAR SEU PROJETO (grátis):
1. Acesse console.firebase.google.com
2. Clique em "Criar um projeto" → dê um nome (ex: op-mesa)
3. Vá em Build → Realtime Database → Criar banco de dados
4. Escolha "Modo de teste" (30 dias, depois ajuste as regras)
5. Vá em Configurações do projeto → Seus apps → Web (</>)
6. Registre o app e copie o objeto firebaseConfig abaixo
